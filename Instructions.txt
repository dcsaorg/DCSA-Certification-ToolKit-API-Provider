Developer instruction
run compatibility tool kit

1.Using Maven
mvn clean compile exec:java -Dexec.mainClass=org.testng.TestNG -Dexec.args="suitexmls\TnTV2-TestSuit.xml"
2. using IDE as TestNG projects
3. Create package using bat file(for window) and run following command from ctk folder under package (e.g. TNT-v2.2.0/ctk)
java -jar DCSA-Validator-Toolkit.jar TestSuite.xml

Create package

run corresponding bat file(for window) to create package folder(e.g. TNT-v2.2.0) under application directory. Which can be zipped and can be shipped.
for another operating system corresponding file can be created to create package folder if required
